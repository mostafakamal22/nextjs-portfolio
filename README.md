## Run Locally


Clone the project


```bash

  git clone https://github.com/mostafakamal22/nextjs-portfolio

```


Go to the project directory


```bash

  cd nextjs-portfolio

```


Install dependencies


```bash

  npm install

```


Start the server


```bash

  npm run start

```
